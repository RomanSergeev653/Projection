# ProjectiON

Веб-приложение для планирования нагрузки по нескольким проектам.

## Возможности

- ✅ Создание и управление проектами
- ✅ Автоматическое распределение часов по дням
- ✅ Ручное редактирование нагрузки конкретных дней
- ✅ Автоматическое перераспределение оставшихся часов
- ✅ Просмотр суммарной нагрузки по дням
- ✅ Индикатор перегрузки (при превышении лимита)
- ✅ Визуализация: Timeline проектов, график нагрузки, календарная сетка

## Установка и запуск

### Быстрый старт

```bash
# Клонировать репозиторий
git clone <repository-url>
cd ProjectON

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

### Для Ubuntu/Linux

См. подробные инструкции в файле [DEPLOY.md](./DEPLOY.md)

### Сборка для продакшена

```bash
# Собрать проект
npm run build

# Запустить preview
npm run preview
```

Собранные файлы в папке `dist/` можно развернуть на любом веб-сервере.

## Использование

1. **Создание проекта**: Нажмите "Создать проект" и заполните форму
   - Название проекта
   - Дата начала
   - Дедлайн
   - Всего часов
   - Цвет проекта

2. **Автоматическое распределение**: После создания проекта часы автоматически распределяются равномерно по дням между датой начала и дедлайном

3. **Ручное редактирование**: В календарной сетке можно изменить нагрузку конкретного дня для любого проекта. Оставшиеся часы автоматически перераспределятся

4. **Настройки**: Установите максимальную дневную нагрузку. Дни с превышением будут помечены как перегруженные

## Технологии

- React 18
- TypeScript
- Vite
- date-fns
- LocalStorage для хранения данных
